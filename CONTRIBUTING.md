# Contributing to SLTC GPA Companion

Thank you for your interest in contributing to the SLTC GPA Companion! This guide outlines how you can contribute to improving this application for the SLTC student community.

## How to Contribute

There are several ways you can help improve this companion app:

1. **Expanding the Course Catalog**: You can contribute by adding missing predefined modules, course codes, and standard credit values for your respective departments and semesters.
2. **Updating Academic Rules**: As university policies evolve (such as credit requirements or GPA safety boundaries), you can submit updates to the rule engine.
3. **Bug Fixes & Feature Enhancements**: Help resolve open issues, improve application responsiveness, or introduce handy utility features.
4. **Documentation**: Help keep documentation up to date and correct any inaccuracies regarding degree structures.

## Submission Guidelines

- **Clean Commit History**: Write clear, descriptive commit messages outlining what was changed.
- **Maintain Design Standards**: Adhere strictly to the Material 3 design system, keeping layouts clean, spacious, and responsive across various screen dimensions.
- **Private Data Security**: Avoid storing any personal or hardcoded sensitive credentials in files or repository branches.
- **Testing**: Verify changes compile successfully before submitting a pull request to ensure the application remains stable.

We appreciate your time and effort in making academic tracking simpler and more accessible for every student!
